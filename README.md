# Some-questions-for-Numerical-Game-Designer
## 1.武器强化问题
  某游戏中有一把武器有0到15的强化等级，每次强化成功率为1-5级70%，6-10级50%，11-15级30%。成功时升1级，失败时降1级，每次操作消耗2个材料。（0级失败后为0级）
  问：一把武器从0级强化到15级平均需要多少材料？
